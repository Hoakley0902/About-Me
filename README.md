# Personal Portfolio Website

A personal portfolio website built with React, JavaScript, HTML, CSS, and Vite. The site showcases my background, technical skills, web development projects, and professional experience as I pursue opportunities in software and web development.

## Live Demo

[View My Portfolio](haley-oakley-portfolio.netlify.app)

## Features

- About Me section introducing my background and interests
- Project highlights showcasing my web development work
- Technical skills and technologies
- Responsive layout for different screen sizes
- Custom styling using CSS
- Component-based interface built with React

## Tech Stack

- React
- JavaScript
- HTML
- CSS
- Vite

I created this portfolio to showcase my development work and continue strengthening my front-end development skills. The site provides a central place for employers and recruiters to learn about my background, view my projects, and explore the technologies I have worked with.
